# Youtube-Comment-Stream
## まず初めに<br>
このソフトはYoutubeライブのコメントを取得し、ニコニコ動画のような流れるコメントをブルーバック(以下BB)の画面に映し出すソフトです。<br>
BBの画面に表示されるので、obsなどの配信ツール上でクロマキー合成等を使うことにより、簡単に流れるコメントを配信画面に合成することができます。<br>
